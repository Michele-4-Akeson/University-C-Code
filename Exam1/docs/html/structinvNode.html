Detailed Description

invNode is a struct containing fields: name, description, value, and next This structure repersents an inventory slot, and the respective name, and description, as well as the value of an item in that slot. It also contains the pointer to the next slot

goblin_loot and orc_loot are global instances of the invNode struct and acts as in game items
